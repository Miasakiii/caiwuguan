# 财务官 - 产品需求文档 (PRD)

## 产品概述

**名称**：财务官
**定位**：Android 端自动记账应用，通过读取支付通知实现零手动记账
**核心理念**：免登录、本地存储、不被杀后台、AI 辅助分析

## 目标用户

- 频繁使用微信/支付宝支付的个人用户
- 想记账但嫌手动记录麻烦的人
- 对隐私敏感、不想把账单上传云端的人

## 功能清单

### P0 - MVP 必做

| 功能 | 说明 |
| ---- | ---- |
| 通知监听 | 读取微信、支付宝、银行 App 的支付通知 |
| 自动记账 | 解析通知内容，自动创建收支记录 |
| 消费分类 | 基于商户名/关键词自动分类（餐饮、交通、购物等） |
| 账单列表 | 按时间展示所有记录，支持手动编辑/删除/补录 |
| 手动记账 | 手动输入金额、分类、商户 |
| 月度统计 | 月收支汇总、分类饼图、趋势折线图 |
| 数据持久化 | Room 本地数据库，数据不丢失 |

### P1 - 第二期

| 功能 | 说明 |
| ---- | ---- |
| 后台保活 | 前台 Service（specialUse 类型）+ WorkManager + 厂商适配引导 |
| **账单导入** | **导入微信/支付宝 CSV 账单，自动去重，批量补录历史数据** |
| 预算管理 | 设置月度预算，超支提醒 |
| 搜索筛选 | 按金额/商户/分类/时间筛选 |
| 数据导出 | CSV 导出 |
| 多账本 | 支持创建多个独立账本 |

### P2 - 第三期

| 功能 | 说明 |
| ---- | ---- |
| AI 分析 | 接入免费 LLM，生成消费洞察和建议 |
| 智能分类优化 | 基于用户修正记录，训练本地分类模型 |
| 周期性账单 | 自动识别房租、订阅等周期性支出 |
| 桌面小组件 | 快速查看今日/本月消费 |
| 短信监听 | 读取银行扣款短信（可选补充通道，Android 4.4+ 受限） |

## 技术约束

- **最低 Android 版本**：API 26 (Android 8.0)
- **目标 Android 版本**：API 34 (Android 14)
- **语言**：Kotlin
- **UI 框架**：Jetpack Compose
- **架构**：MVVM + Clean Architecture
- **本地存储**：Room (SQLite)
- **最低设备**：2GB RAM，100MB 存储
